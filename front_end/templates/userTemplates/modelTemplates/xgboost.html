XGBoost Analysis

  • Non-linearity: XGBoost can model non-linear relationships without explicit feature transformations.
  • Performance: XGBoost is optimized for speed and performance, which is crucial when working with a large dataset.
  • Outliers: XGBoost is less sensitive to outliers due to the boosting technique used in its training.
  • Boosting Model: Trains decision trees sequentially, where each new tree attempts to correct the errors of the previous tree. Helps reduce bias.
  • Model Interpretability: XGBoost returns a feature importance score making it easy to see which features influence the model.
