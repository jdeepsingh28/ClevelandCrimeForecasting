Neural Network Analysis

  • Chosen Model: Combination of Bidirectional LSTM (Long-Term Short Memory) layers for temporal data and an embedding layer for spatial data.
  • LSTM networks are designed to handle sequence data and are effective in capturing temporal dependencies.
  • By using Bidirectional LSTM, sequences are processed both forward and backward, allowing the model to use both past and future contexts to make decisions.
  • By using an embedding layer, the model is better able to learn relationships between census blocks.
  • By combining both pathways, the model can leverage dependencies learned for temporal and spatial features when making predictions.
